# HDFS Maintenance Toolkit

A collection of scripts for managing and maintaining HDFS health in production environments.

## Scripts
- `hdfs_balancer.sh`: Automates HDFS periodic rebalancing.
- `cleanup_logs.py`: Purges old application logs to save space.
- `check_health.sh`: Quick health check for NameNodes.

## Requirements
- Python 2.7+
- Hadoop Client
